# To-Do Life Dashboard

Welcome to the **To-Do Life Dashboard**, a personal, homepage-style single-page web application designed to help you organize your day right from your browser. 

This project was built entirely with Vanilla HTML, CSS, and JavaScript—meaning no frameworks, no bundlers, no backend, and no build step. All your data is kept private and securely stored in your browser's local storage.

---

## Features

The dashboard consists of four core modules and a customizable interface:

1. 🌤️ **Greeting Module**: Displays a live clock, current date, and a time-aware personalized greeting.
2. ⏱️ **Focus Timer**: A 25-minute Pomodoro countdown timer equipped with Start, Stop, and Reset controls to help you stay focused.
3. 📝 **To-Do List**: A full-featured task manager. Add, edit, complete, delete, and sort your tasks to keep track of your daily goals.
4. 🔗 **Quick Links**: Save your most visited URLs as handy shortcut buttons for instant access.
5. 🌓 **Theme Toggle**: Switch between a sleek Light Mode and a rich Dark Mode. The application will remember your preference!

## How to Use

Since this is a client-side only application, getting started is extremely simple:

1. **Clone or Download** the repository.
2. Open `index.html` directly in any modern web browser (Chrome, Firefox, Edge, Safari).
3. Start organizing your day! Your tasks, links, theme preferences, and name will be automatically saved in your browser's Local Storage.

### Running the Tests (For Developers)
If you'd like to contribute or review the automated test suite:
1. Ensure you have Node.js installed.
2. Run `npm install` to install testing dependencies.
3. Run `npx vitest run` to execute the full suite of 124 passing tests.
